## Описание приложения
Приложение предлагает купить тур по определённой цене двумя способами:
1. Обычная оплата по дебетовой карте.
2. Выдача кредита по данным банковской карты.

![](https://github.com/netology-code/aqa-qamid-diplom/raw/main/pic/service.png)

Само приложение не обрабатывает данные по картам, а пересылает их банковским сервисам:
* Сервис платежей
* Кредитный сервис

Приложение в собственной СУБД сохранять информацию о том, успешно ли был совершён платёж и каким способом. Данные карт сохранять не допускается.

## Процедура запуска автотестов
1. Клонировать репозиторий командой: *git clone*
2. Открыть проект в **IntelliJ IDEA**
3. Запустить **Docker** командой: *docker compose up*
4. Запустить **SUT** командой: *java -jar ./artifacts/aqa-shop.jar*
5. Запустить прилодение в Google Chrome по адресу http://localhost:8080
6. Запустить тесты командой: *./gradlew test "-Dselenide.headless=true" --info  *

## Процедура запуска отчета
1. В терминале ввести команду: *./gradlew allureReport*
