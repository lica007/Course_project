## Общие рекомендации
* Для тестирования сервиса не хватает документации
* Отсутствие проверки обязательности полей API
* Отсутствие валидации API
* Привести коды ответов HTTP в соответствие со стандартами REST
* В поле "Владелец" отсутвует валидация: настроить максимальную и минимальную длину, настроить автоматическое переведение данных в верхний регистр, запретить ввод данных кириллицей
* Поправить поле "Год" для ввода срока действия карты до 10 лет от текущего года
* Настроить маскировку для поля CVC\CVV
